Design a modern self-service photo booth website UI for Ignos Studio.

The interface will be used on a large touchscreen kiosk where users select templates and print their photos after a photoshoot session.

Use a Neo-Brutalist UI design style (also known as Bold Outline / Thick Border UI) with a clean, modern layout inspired by Apple product simplicity but with bold brutalist aesthetics.

The interface must feel:

Bold

High contrast

Playful but structured

Highly visible on a large touchscreen

Extremely easy to interact with

Design Style

Use the following Neo-Brutalist UI characteristics:

UI characteristics:

Thick bold borders (3–5px)

High contrast UI elements

Flat surfaces

Strong geometric shapes

Sharp visual hierarchy

Minimal shadows

Large rounded corners

Playful layout but structured

Avoid:

Glassmorphism

Blur effects

Soft gradients

Subtle transparency

Instead focus on bold UI components with strong outlines.

Color System (Monochrome Theme)

Use a strict monochrome color palette.

Primary colors:

Pure black

Charcoal gray

Light gray

White

Background:

Use clean white or light gray background to maximize contrast.

UI Components:

White cards with thick black borders

Black buttons with white text

White buttons with black borders

Minimal color usage

The UI should feel bold, modern, minimal, and highly readable.

Typography

Use modern sans-serif typography.

Recommended styles similar to:

Inter

SF Pro

Space Grotesk

Typography style:

Bold headings

High contrast text

Large font sizes for touch UI

Clear hierarchy

Avoid thin or overly decorative fonts.

Layout Requirements

The interface must be optimized for landscape touchscreen kiosk UI.

Design principles:

Large tap areas

Big spacing

Clear layout structure

Highly visible interactive elements
